# ☕ Shot Up — 카페 키오스크

터치스크린 카페 키오스크 UI 포트폴리오 프로젝트.
매장/포장 선택, 한영 언어 전환, 메뉴 주문, 장바구니, 결제까지 실제 키오스크와 동일한 전체 플로우를 구현했습니다.

## 데모

🔗 **[kiosk-bice-ten.vercel.app](https://kiosk-bice-ten.vercel.app)**

## 주요 기능

| 기능 | 설명 |
|------|------|
| 🏠 홈 화면 | 매장 / 포장 선택 후 주문 시작 |
| 🌐 한영 전환 | 홈 화면에서 언어 선택 → 전체 UI 즉시 전환 |
| 🖼️ 메뉴 이미지 | 카테고리별 실제 메뉴 사진 표시 |
| 🛒 장바구니 | 수량 조절, 삭제, 전체 취소 |
| ☕ 샷 추가 | 커피류 한정, 최대 2샷, 샷당 +500원 |
| 💳 결제 | 신용카드 / 삼성페이 선택 |
| ✅ 주문 완료 | 주문 번호 표시, 5초 후 홈 자동 복귀 |

## 화면 플로우

```
홈 (매장/포장 + 언어 선택)
  └─▶ 메뉴 (카테고리 사이드바 + 그리드 + 장바구니)
        └─▶ 주문 확인
              └─▶ 결제 방법 선택
                    └─▶ 주문 완료 ──▶ 홈
```

## 기술 스택

- **React** + **Vite**
- **Zustand** — 장바구니 / 언어 상태 관리
- **CSS Variables** — 다크 테마 디자인 시스템
- **Vercel** — 배포

## 폴더 구조

```
src/
  features/
    home/       # 홈 화면 (매장/포장 선택, 언어 선택)
    menu/       # 메뉴 데이터, 카테고리 사이드바, 메뉴 그리드
    cart/       # Zustand store, 장바구니 패널, 개별 아이템
    order/      # 주문 확인, 결제 선택, 주문 완료
    i18n/       # 번역 데이터 (한국어/영어), 언어 store
public/
  menu/         # 메뉴 이미지 (20종)
```
